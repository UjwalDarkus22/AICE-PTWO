Exam Tips
Key strategies for answering AICE Psychology questions
📐
Always define before evaluating
When asked to evaluate a study, define the key concept first (e.g. ecological validity, reliability) before applying it to the study.
⚖️
Strengths AND weaknesses
For evaluation questions, give both sides. Link each point directly back to the study — don't make generic claims.
🎯
IV → DV linkage
Always be able to state the IV and DV precisely for each study. Know how each was operationalized (measured).
🔬
Know the controls
Examiners love asking about controls. For each study, know 2–3 specific control measures and why they matter.
🧬
Nature vs. nurture links
Hassett, Baron-Cohen, and Holzel all connect to this debate. Be ready to explain which side each study supports and why.
📊
Quant vs. qual data
Know whether each study produced quantitative, qualitative, or both types of data — and be able to explain the difference and give an example from the study.
🏛️
Ethics checklist
For every study, run through: informed consent, deception, right to withdraw, harm, debriefing, confidentiality. Milgram and Piliavin break almost all of them.
🌍
Generalisation matters
Discuss sample composition (age, sex, culture, size) when evaluating generalisability. A sample of 9 adults (Dement) or one child (Saavedra) is very limited.
